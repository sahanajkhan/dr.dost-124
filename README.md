# 🩺 Dr. Dost – Smart Healthcare & Student Wellness Platform

Dr. Dost is an AI-powered healthcare and wellness platform designed to connect **patients, doctors, and students** in a single ecosystem.
The platform helps users find nearby doctors, access emergency ambulance services, monitor wellness, and interact with an AI assistant for health guidance.

---

## 🚀 Features

### 👨‍⚕️ Doctor–Patient Connection

* Search and connect with nearby doctors
* View doctor profiles and availability
* Book appointments online

### 🗺️ Google Maps Integration

* Locate doctors and hospitals near the user
* Real-time location-based search

### 🚑 Smart Ambulance Dispatch

* Emergency ambulance request system
* Automatic notification to nearby emergency services

### 🤖 AI Chatbot Assistant

* 24/7 health assistant
* Answers basic health queries
* Suggests wellness activities and stress relief tips

### 🧠 Emotion & Wellness Monitoring

* Detects user emotions through facial expression analysis
* Suggests music, videos, or relaxation activities

### 👥 Peer Group Community

* Students can connect with peers
* Ask doubts and form learning groups
* Skill-based community discussions

### 📅 Smart Calendar

* AI-powered schedule optimization
* Suggests study time and relaxation time

### 🎥 Health & Wellness Content

* Curated health videos
* Mental wellness and fitness suggestions

## ⚙️ Installation

### 1️⃣ Clone the repository

```
git clone https://github.com/yourusername/dr-dost-platform.git
```

### 2️⃣ Go to project folder

```
cd dr-dost-platform
```

### 3️⃣ Install dependencies

Frontend

```
cd frontend
npm install
```

Backend

```
cd backend
npm install
```

---

## ▶️ Run the Project

Start backend

```
npm start
```

Start frontend

```
npm run dev
```
